# mad-converter

Moroccan local converter to make it easy for your everyday life.

Converts between:
- MAD (Dirham)
- Ryal (local unit): 1 MAD = 20 ryal
- Franc (local unit): 1 MAD = 100 franc

## Dev

```bash
npm install
npm run dev
```

Then open:
- http://localhost:3000/ar
- http://localhost:3000/fr
- http://localhost:3000/en

## SEO base URL (recommended for production)

Set:
- `NEXT_PUBLIC_SITE_URL=https://mad-converter.wellycloud.com`

This is used for sitemap and canonical URLs.
